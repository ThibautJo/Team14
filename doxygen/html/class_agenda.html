◆ index()

index ( )

Haalt alle activiteiten van de aangemelde zwemmer op en toont deze op het scherm in een agenda.

De verschillende soorten activiteiten worden via aparte methoden ingeladen en omgevormd tot één grote array. Deze array wordt omgezet in een JSON object dat wordt ingeladen in de agenda.
